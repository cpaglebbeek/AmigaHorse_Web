BASIC Asset-Setup

Eenmalige setup. Lever drie bestanden die je legaal in bezit hebt (Cloanto Amiga Forever of eigen rip):

  1. Stap 1 — Kickstart 1.3 ROM

    Bestand kick13.rom, verwacht ±256 KB.

  2. Stap 2 — Workbench 1.3 ADF

    Bestand wb13.adf, verwacht exact 880 KB.

  3. Stap 3 — AmigaBASIC binary (optioneel)

    Skip deze stap normaal — AmigaBASIC zit al in de Workbench 1.3 ADF uit stap 2 en wordt automatisch geopend tijdens de bake-flow. Upload hier alleen als je een aparte/aangepaste AmigaBASIC-binary hebt.

  4. Stap 4 — Warm-snapshot bake

    Eenmalige boot van de Amiga + start AmigaBASIC + save-state-freeze (~10 sec).

Let op: AmigaBASIC werkt alléén op Kickstart 1.x (Commodore-incompatibility op KS 2.0+). Géén binaries worden naar GitHub gepushed — alles blijft in je browser (IndexedDB).
